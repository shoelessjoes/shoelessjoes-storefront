## How these get onto the site
image_picker fields in the theme pull from Shopify **Content > Files**, not the repo.
1. Upload all sj-*.jpg to Shopify Files (bulk drag-and-drop).
2. In the theme editor, set each section's image to the matching file above.
(The sandbox can't push to Shopify Files directly — network is locked down.)
